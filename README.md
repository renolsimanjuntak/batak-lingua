# Batak Lingua - Transliterasi Aksara Batak

**Batak Lingua** adalah aplikasi web modern berbasis JavaScript untuk mengubah teks Latin menjadi Aksara Batak secara otomatis (*transliterasi*). Proyek ini dibuat untuk melestarikan budaya dan mempermudah generasi muda dalam mempelajari aksara Batak.

---

## ✨ Fitur Utama
* **Akurasi Tinggi:** Menggunakan pemetaan (*mapping*) karakter yang disesuaikan dengan aturan penulisan aksara.
* **Multi-Dialek:** Mendukung dialek Toba (Simalungun, Angkola, Karo, dan Pakpak dalam pengembangan).
* **Antarmuka Modern:** Desain responsif yang nyaman digunakan di HP maupun Komputer.

---

## 🚀 Cara Penggunaan
1.  Buka [Batak Lingua](https://renolsimanjuntak.github.io/batak-lingua/)
2.  Pilih dialek pada menu navigasi (Default: **Toba**).
3.  Ketik kalimat latin pada kotak input yang tersedia.
4.  Aksara Batak akan muncul secara otomatis di kotak output.
5.  Klik tombol **Salin Aksara** untuk menyalin hasilnya.

---

## 📂 Struktur Proyek
* `index.html` - Struktur utama dan antarmuka pengguna.
* `src/` - Berisi src transliterasi. 
* `data/` - Berisi database pemetaan karakter.

# Batak Lingua - Batak Script Transliteration

**Batak Lingua** is a modern JavaScript-based web application designed to automatically convert Latin text into Batak Script (*transliteration*). This project was created to preserve cultural heritage and make it easier for the younger generation to learn the Batak script.

---

## ✨ Key Features
* **High Accuracy:** Uses character mapping adjusted to traditional script writing rules.
* **Multi-Dialect:** Supports the Toba dialect (Simalungun, Angkola, Karo, and Pakpak are under development).
* **Modern Interface:** Responsive design, comfortable for both Mobile and Desktop use.

---

## 🚀 How to Use
1.  Open [Batak Lingua](https://renolsimanjuntak.github.io/batak-lingua/)
2.  Select a dialect from the navigation menu (Default: **Toba**).
3.  Type your Latin sentences in the provided input box.
4.  The Batak Script will appear automatically in the output box.
5.  Click the **Copy Script** button to copy the result.

---

## 📂 Project Structure
* `index.html` - Main structure and user interface.
* `src/` - Contains the transliteration source code.
* `data/` - Contains the character mapping database.
